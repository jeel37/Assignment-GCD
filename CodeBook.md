# Code Book For run_analysis.R

This Document describes the variables, procedure and data


## VARIABLES

### Data Frames

* activity
* AvgVals
* name
* stest
* strain
* x
* x2
* x3
* xtest
* xtrain
* ytest
* ytrain

### Character

* activitylabel
* mnames

### Integer

* bothcol: Contains the column numbers of the columns that contains 'Mean' or 'Std'

### Other

* fileUrl: [URL](https://d396qusza40orc.cloudfront.net/getdata%2Fprojectfiles%2FUCI%20HAR%20Dataset.zip) of the data being used for the project 



## PROCEDURE
* Download File and unzip it
* Set Working Directory
* Read:
  * Training Data and Testing Data
  * Training and Testing Labels
  * Training and Testing Subjects
* Combine training and testing data, labels and subjects
* Merge the training and the test sets to create one data set.
* Find and apply Column Labels
* Extract only the measurements on the mean and standard deviation for each measurement.
* Use descriptive activity names to name the activities in the data set
* Appropriately labels the data set with descriptive variable names.
* Reorder Subjects and Activities
* Creating independent tidy data file 
  * Use aggregate function to find average of each subject and each activity
  * Change column names as needed
  * Write the table out to a txt file
